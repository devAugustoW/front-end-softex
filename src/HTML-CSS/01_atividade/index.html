2. Como o HTML se relaciona com outras tecnologias da web, como o CSS e o JavaScript? Como essas três tecnologias trabalham em conjunto para criar páginas web interativas e estilizadas?

HTML cria a estrutura: Define os elementos da página. CSS estiliza a estrutura: Aplica estilos aos elementos HTML, definindo a aparência visual. JavaScript manipula a estrutura e os estilos: Adiciona interatividade, modifica o conteúdo e os estilos da página em tempo real, em resposta às ações do usuário.

      A Importância das Tags de Listagem: 'ul' e 'ol'. As tags 'ul' e 'ol' são fundamentais para organizar e apresentar informações de forma clara e eficiente em uma página HTML. Elas fornecem uma estrutura visual que facilita a leitura e a compreensão do conteúdo.

      'ul' (Unordered List): Lista Desordenada Uso: Empregada quando a ordem dos itens não é relevante. Características: Os itens são geralmente marcados com marcadores (bullets) por padrão. A ordem dos itens pode ser alterada sem afetar o significado do conteúdo. Exemplos: Lista de ingredientes de uma receita Lista de itens a serem comprados Lista de links para páginas relacionadas


      'ol' (Ordered List): Lista Ordenada Uso: Utilizada quando a ordem dos itens é significativa. Características: Os itens são numerados, indicando uma sequência específica. A alteração da ordem dos itens pode mudar o significado do conteúdo. Exemplos: Passos de um tutorial Lista de etapas de um processo Ranking de produtos



      5. Como a tag 'a' é utilizada para criar links em HTML? Explique como você pode vincular para outras páginas, para arquivos locais e criar âncoras dentro de uma página.

      A tag 'a' permite a navegação entre páginas diferentes, o download de arquivos e o acesso a seçoes específicas dentro da página. 'a href="link" 'Texto do link'/a' 'a': A tag de âncora que define o início e o fim do link. href: O atributo que especifica o endereço (URL) para onde o link irá direcionar. Texto do link: O texto visível que o usuário clica para seguir o link.



      6. Além do atributo href, quais outros atributos importantes a tag pode ter? Como o atributo target pode ser usado para controlar como o link é aberto pelo navegador?

      title: Adiciona um texto de dica que aparece quando o mouse passa sobre o link, fornecendo mais informações sobre o destino. rel: Especifica a relação entre a página atual e o recurso linkado. É utilizado principalmente para SEO e para indicar o tipo de relacionamento entre as páginas (por exemplo, nofollow para links que não devem ser seguidos por mecanismos de busca). download: Indica que o recurso vinculado deve ser baixado quando clicado, ao invés de ser aberto no navegador. ping: Permite notificar URLs específicas quando um link é clicado, útil para rastrear cliques em links. hreflang: Especifica a língua do recurso vinculado, utilizado para SEO em sites multilíngues.



      7. Descreva a diferença entre os atributos src e alt em relação à tag 'img'. Por que é importante fornecer um atributo alt adequado para imagens?

      O atributo src tem relação com o caminho de onde a imagem se encontra. O atributo alt tem relação com a acessibilidade da tag, netse caso da imagem. Para pessoas que tem problemas de visão e o alt é o seu guia. O que for digitado neste atributo a página irá ler em voz alta para o usuário. Por isso ter as especificações do alt bem escritas é essencial para este enorme grupo de pessoas.

      9. Embora tabelas sejam usadas para exibir dados tabulares, às vezes é recomendado evitá-las em favor de outras formas de apresentação de dados. Por quê? E quais são os cenários em que as tabelas são apropriadas?

      As tabelas podem apresentar algumas desvantagens: Acessibilidade: Tabelas complexas podem ser difíceis de serem interpretadas por leitores de tela e usuários com deficiência visual. Responsividade: Tabelas podem se tornar quebradas e difíceis de visualizar em diferentes tamanhos de tela, especialmente em dispositivos móveis. SEO: Tabelas complexas podem dificultar a indexação dos dados por mecanismos de busca. Design: Tabelas podem limitar a flexibilidade no design da página, tornando a aparência menos atraente. Performance: Tabelas grandes e complexas podem afetar o tempo de carregamento da página.

      Cenários em que as tabelas são apropriadas: Dados numéricos: Tabelas são ideais para apresentar dados numéricos de forma organizada e comparativa. Dados estruturados: Quando os dados possuem uma estrutura clara de linhas e colunas, as tabelas facilitam a compreensão das relações entre os dados. Comparação de dados: Tabelas são excelentes para comparar diferentes conjuntos de dados lado a lado. Tabelas de preços: Tabelas são comumente usadas para apresentar preços de produtos ou serviços de forma organizada. Tabelas de dados técnicos: Dados técnicos, como especificações de produtos, são frequentemente apresentados em tabelas.
